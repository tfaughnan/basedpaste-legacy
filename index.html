NAME
        basedpaste - a highly based pastebin and link shortener

SYNOPSIS
        curl -F file=@$FILE $BASEDPASTE_URL
        curl -F url=$URL $BASEDPASTE_URL

DESCRIPTION
        basedpaste is a web service that hosts files and shortens links, with
        an emphasis on simplicity. You are currently viewing the instance
        hosted at {{.Url}}. Its maximum upload size is {{.MaxFileMiB}} MiB.

OPTIONS
        Make a POST request with either 'file' or 'url' as a form-field. Use
        multipart/form-data, not application/x-www-form-urlencoded. See the
        documentation for curl(1) or your preferred HTTP client.

        The administrator of a basedpaste instance may optionally require
        an API key to POST. If this is the case, users must add an 'auth'
        form-field with their key to requests. This instance {{if .RequireAuth}}requires{{else}}does not require{{end}}
        authentication.

EXAMPLES
        curl -F file=@/tmp/logs.txt {{.Url}}
        curl -F file=@<(make all) {{.Url}} 
        curl -F file=@~/.vimrc -F auth='2ab96390c7dbe3439de74d0c9b0b1767' \
                {{.Url}}        # if API key required
        curl -F url='https://www.youtube.com/watch?v=dQw4w9WgXcQ' \
                {{.Url}}
